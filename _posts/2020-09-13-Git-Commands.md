## Git Commands

## **A_**

 (1) git add

to stage specific changes only
```
$ git add [directory]
```

to add all
```
$ git add *
```

## **F_**

 (1) git fetch


## **I_**

 (1) git init

initialize a new git repository + create .git folder

```
$ git init
```


## **P_**

 (1) git pull


## **U_**

 (1) Username

here are ways to check git username

```
$ git config user.name
```

```
$ git config --list
```

```
# go to HOME directory
# then
$less .gitconfig
```
