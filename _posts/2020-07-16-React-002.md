## React Hooks!

> Hooks 는 클래스를 사용하지 않고 함수형 컴포넌트에서도
> 
> 상태관리를 가능하게 해줍니다.

### (1) useState
```js
import React, { useState } from 'react';

function Counter() {
    const [count, setCount] = useState(0);
}
// count 는 현재상태를, setCount 는 그 상태를 업데이트 할 때 사용 할 function 을 의미합니다.

// useState(초기값/상태의 기본값)
```
